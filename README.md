# AI-Powered Cybersecurity Threat Detection System

## Overview
AI-based intrusion detection system that identifies network anomalies using machine learning.

## Problem
Manual monitoring is slow and inefficient.

## Solution
ML-based automated threat detection system.

## Tech Stack
Python, Scikit-learn, Pandas, Isolation Forest

## Dataset
UNSW-NB15 / CICIDS2017

## Features
- Anomaly detection
- Attack classification
- Visualization

## How to Run
pip install -r requirements.txt
python main.py

## Results
- Detects anomalies with high accuracy
